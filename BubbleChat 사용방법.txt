● 작업 환경
: 이클립스, 안드로이드 스튜디오

● 실행 방법
00. 이클립스에서 Java Build Path에 jgoodies-common-1.8.1.jar 파일과 jgoodies-forms-1.9.0.jar 파일을 추가해주어야함

0. ClientBackgroound.java 파일의 connet 함수에서
socket = new Socket("192.168.0.23", client_port);
부분의 IP주소를 현재 노트북/핸드폰이 사용 중인 와이파이의 IP주소로 변경 (cmd에서 ipconfig)
(이 부분은 고정 IP 할당 후 그 IP로 수정하기)
- PC버전, 안드로이드 버전 두 가지 모두 변경해주어야 함

1. 이클립스에서 ServerGui.java 실행
port 번호 지정 - 5000~65000 사이 아무거나 (포트 번호 = 채팅방 번호)
- ServerGui.java를 계속 실행시켜서 새로운 채팅방(=포트) 여러 개 개설 가능

2. 이클립스에서 ClientGui.java 실행
4자리 이하 닉네임 아무거나 입력
들어가고 싶은 채팅방의 포트 번호 입력
- ClientGui.java를 계속 실행시켜서 새로운 PC 클라이언트 여러 개 생성 가능

3. 안드로이드 스튜디오에서 앱 실행
4자리 이하 닉네임 아무거나 입력
들어가고 싶은 채팅방의 포트 번호 입력
버튼 클릭해서 채팅방 입장

4. 기능
서버에서 메시지를 보내면 파란색 버블로 표시됨
내가 보낸 메시지는 노란색 버블로 표시됨
다른 사용자가 보낸 메시지는 흰색 버블로 표시됨
다른 사용자가 보낸 버블을 더블클릭 하면 좋아요 표시를 할 수 있음
사용자가 나가고 싶을 때 상단의 exit 버튼을 클릭하면 나갈 수 있음
서버 GUI에서 포트 닫기 버튼 클릭하면 해당 채팅방 사라짐(=포트 닫힘)
열려있는 포트 번호 리스트는 portlist.txt 파일에서 쉼표로 구분하여 관리됨 (포트 닫기 버튼 클릭시 자동으로 지워짐)



● 개선할 점
0. 고정 IP 할당

3. 서버 GUI 디자인 하기
4. 사소한 오류 - '채팅방에 입장하였습니다'를 더블클릭 해도 좋아요가 되는 것 막기
